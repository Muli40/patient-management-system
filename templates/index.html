About the System

The Patient Management System is designed to simplify healthcare data handling by allowing hospitals and clinics to manage patient and doctor records digitally. It improves accuracy, saves time, and enhances patient care by centralizing all essential information in one secure platform.

This system was built using HTML, CSS, and Flask (or PHP/MySQL) for seamless integration between the user interface and database operations.
